### Create a Basic JavaScript Object
```js
let dog = {
  name: "You",
  numLegs: 4
};
```

### Use Dot Notation to Access the Properties of an Object
```js
let dog = {
  name: "Spot",
  numLegs: 4
};
// Only change code below this line
console.log('Dog Name:'+dog.name, 'Num of Legs:'+dog.numLegs)
```
